Vanilla javaScript

JavaScript is a versatile programming language primarily used for creating interactive and dynamic content on websites, such as animations, form validation, and real-time updates. It runs in the browser and on servers (using environments like Node.js), making it essential for modern web development.

React

React is a popular JavaScript library for building user interfaces, developed by Facebook. It allows developers to create reusable components and manage complex application states efficiently with a virtual DOM for fast updates.

Node.js

Node.js is a JavaScript runtime that allows developers to run JavaScript code on the server side. It is built on Chrome's V8 engine and is widely used for building scalable, fast, and lightweight backend applications.

MongoDB

MongoDB is a NoSQL database that stores data in a flexible, JSON-like format called documents, making it ideal for handling unstructured or semi-structured data. It is designed for scalability and high performance, often used in modern web applications to manage large datasets efficiently.
